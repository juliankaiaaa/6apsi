# React Theory

The *why* behind React and modern front-end design. The
[`m2-react/`](../m2-react/) reading teaches you *how* to write React; this
library explains the ideas underneath it: what React actually is, how it renders,
how to architect components, and how professionals organize and design real
interfaces.

**You do not need to read all of these up front.** They are **reference**: each
activity points you to the one or two pages you need, right when you need them.
They are written to be read in order if you want the full picture, but dipping
in is completely fine.

**Want a head start before Module 2, Activity 1?** Read just these three:
**01** (what React is), **02** (the Virtual DOM), and **03** (JSX and
components). Pull in the rest as the activities send you to them.
